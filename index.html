Atividade SA3-ETAPA-01

Jackson Douglas de Souza

Você deverá criar um algoritmo em pseudocódigo, com as seguintes especificações: Implementar uma estrutura de repetição que solicite as informações do usuário até que seja informado o número três. A cada repetição, apresentar um menu com as seguintes opções: 1º) Cadastrar novos Usuários; 2º) Listar todos os usuários cadastrados; 3º) Sair do sistema. Quando o cliente digitar o número 1, solicitar as informações dele: nome e idade; Ao ser digitado o número 3, o programa deve encerrar o laço de repetição.
